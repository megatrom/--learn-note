## photoshop
> 转用于图片设计和处理的软件.一般在工作中,前端的ui设计稿都是有photoshop设计好的,格式为xxx.psd文件.

### 安装
+ 自行安装.

### 常用操作
+ 常用功能
	- 窗口==> 图层/信息/选项/工具
	- 矩形选择框 用于测量尺寸  
+ ctrl+滚轮  放大缩小
+ space 拖拽
+ psd中文字上方 快捷键t选中文字
+ ctrl+shift+c  复制选取图片   
+ ctrl+n  创新新画布 跟刚才的选区大小一致
+ ctrl+v  粘贴

## 图片
+ jpg  一般用于照片保存,适合色彩绚丽的图片
+ png  一般用于小图标,适合保存色彩相对单一的图片,而且背景可以透明
